# Abaqus 批量计算工具 更新说明
## 版本兼容说明
本工具**不局限于兼容 Abaqus 2024**，支持多版本 Abaqus 匹配：
- `abq2024` → Abaqus 2024 版
- `abq6145` → Abaqus 6.14.5 版
- `abq6144` → Abaqus 6.14.4 版

---

## 核心功能更新
### 1. CPU核心数输入功能
- 在**顶部工具栏**新增CPU核心数输入框
- 默认配置：4核心
- 功能细节：
  - 开始任务时自动读取输入值并完成合法性验证
  - 任务提交时，日志中实时显示当前使用的CPU核心数

### 2. 停止当前任务功能
- 新增**橙色「停止当前任务」**按钮
- 执行逻辑：停止当前正在计算的任务 → 自动继续执行下一个任务
- 实现方式：通过 `taskkill` 命令（Windows系统）强制终止Abaqus进程
- 日志记录：自动记录任务停止的相关信息

### 3. 终止所有任务功能
- 新增**红色「终止所有任务」**按钮
- 执行逻辑：终止当前任务 + 取消所有未执行任务 → 彻底结束批量计算流程
- 安全机制：弹出确认提示框，防止误操作
- 日志记录：完整记录终止操作信息

---

## 功能核心区别
| 功能 | 作用范围 | 执行结果 |
| :--- | :--- | :--- |
| **停止当前任务** | 仅当前运行任务 | 中断当前计算，自动执行下一个任务 |
| **终止所有任务** | 全部任务（当前+未执行） | 完全结束批量流程，不执行任何后续任务 |

---

## 新增特性与优化
1. **界面优化**：窗口尺寸调整为 `1000x750`，适配更多控件展示
2. **进程跟踪**：新增 `current_process` 变量，实时跟踪当前运行的进程
3. **状态区分**：新增 `should_terminate` 标志，精准区分「停止」和「终止」操作
4. **资源管理**：增强进程管理逻辑，确保进程资源正确清理，无残留

### 总结
1. 工具兼容**多版本Abaqus**，自动匹配版本标识；
2. 新增三大核心功能：CPU核心自定义、单任务停止、全任务终止；
3. 优化界面、进程管理与状态区分，提升批量计算稳定性和操作体验。
