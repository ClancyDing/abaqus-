不只兼容ABAQUS2024
abq2024  ==》 abaqus 2024 版
abq6145  ==》 abaqus 6.14.5 版
abq6144  ==》 abaqus 6.14.4 版
------------------------------

1. CPU核心数输入功能
在顶部工具栏添加了CPU核心数输入框
默认值为4核心
在开始任务时会自动读取并验证输入值
任务提交时会显示使用的CPU核心数
2. 停止当前任务功能
新增橙色"停止当前任务"按钮
停止当前正在计算的任务后，自动继续执行下一个任务
通过taskkill命令（Windows）强制终止Abaqus进程
日志中会记录停止信息
3. 终止所有任务功能
新增红色"终止所有任务"按钮
终止当前任务并取消剩余所有未执行的任务
彻底结束批量计算流程
弹出提示框确认终止操作
功能区别：
停止当前任务：只中断当前正在运行的计算，然后继续下一个
终止所有任务：完全结束整个批量计算流程，不再执行任何后续任务
新增特性：
窗口大小调整为1000x750以容纳更多控件
添加了current_process变量跟踪当前运行的进程
添加了should_terminate标志区分停止和终止操作
增强的进程管理，确保正确清理进程资源
